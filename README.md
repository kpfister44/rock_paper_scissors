# rock_paper_scissors
Rock paper scissors game which can be played through the browser. No styling has been added. 

What I learned

how to use use js to manipulate the DOM (create elements and classes, display text, use events)
how to store the 'click' addEventListener in a variable which can be later used
how to utilize callback arguments on addEventListener
how to call functions inside other functions
the importance of return values for functions
how parameters and arguments work when calling functions
the importance of knowing the difference between = , == , and ===
